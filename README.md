# 전자정부프레임워크 Batch Custom

## 1. 개요
- 전자정부프레임워크(이하 egov) 3.10을 활용해 Batch 기능을 구현한다.
- 기존 egov에서 xml 위주의 설정을 java 위주의 설정으로 변경한다.
- Scheduler를 활용하여 정해진 시간과 필요시 즉시 실행이 가능한 구조로 구현한다.

## 2. 구성
- springframework 4.3.25
- egovframework 3.10
- mybatis 3.4.6
- cubrid 11.2
